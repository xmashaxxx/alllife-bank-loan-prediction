# Dataset

**Name:** AllLife Bank Personal Loan Dataset
**Source:** Provided as part of MIT IDSS coursework (not redistributable)
**Size:** 5,000 rows x 14 columns

## Key Columns

| Column | Description |
|--------|-------------|
| Age | Customer age in years |
| Experience | Years of professional experience |
| Income | Annual income in $000s |
| Family | Family size |
| CCAvg | Monthly credit card spending in $000s |
| Education | 1=Undergrad, 2=Graduate, 3=Advanced |
| Mortgage | Mortgage value in $000s |
| Personal Loan | Target: 1=accepted loan offer, 0=did not |
| Securities Account | Has securities account (1/0) |
| CD Account | Has certificate of deposit account (1/0) |
| Online | Uses online banking (1/0) |
| CreditCard | Has credit card from AllLife Bank (1/0) |
